Конвертер	Что делает	Пример результата
!s	str(obj)	hello
!r	repr(obj)	'hello'
!a	ascii(obj)	'\u043f\u0440\u0438\u0432\u0435\u0442'

!r — вызывает repr(obj)
Возвращает формальное представление объекта.
Используется для отладки, логов.
Строки будут в кавычках.

!s — вызывает str(obj)
Возвращает человеко-читаемое представление.
Используется при выводе пользователю.


!a — вызывает ascii(obj)
Как repr, но экранирует не-ASCII символы, делает вывод безопасным для ASCII-сред.
